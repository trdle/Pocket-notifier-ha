# Brand icons

These are the source icons for the PocketNotifier integration's logo in
HACS and the Home Assistant integrations page.

Home Assistant/HACS do **not** read icons from this repository — they are
served from the central [`home-assistant/brands`](https://github.com/home-assistant/brands)
repo. To make the icon appear, submit these files to that repo:

| File here | Destination in `home-assistant/brands` |
| --------- | -------------------------------------- |
| `icon.png` (256×256) | `custom_integrations/pocketnotifier/icon.png` |
| `icon@2x.png` (512×512) | `custom_integrations/pocketnotifier/icon@2x.png` |

After that PR is merged, the icon is available at
`https://brands.home-assistant.io/pocketnotifier/icon.png`.
